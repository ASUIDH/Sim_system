# Malware-Simlarity
Windows恶意软件相似性度量系统

框架：vue2.0+flask+mysql



## Overview
The repository is organised as follows:

- `fe/` 系统前端
- `be/` 系统后端

## Run
前端：在终端运行
``` 
 npm run serve
```
后端：在终端运行
``` 
python manage.py runserver 
```
数据库更新：
``` 
python manage.py db init

python manage.py db migrate -m "init"

python manage.py db upgrade
```
